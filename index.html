Objetivos do grupo na Disciplina

Durante a disciplina, o nosso grupo aprenderá sobre HTML, CSS, JavaScript e PHP. O nosso objetivo é aprender esses tópicos e conseguir aplicá-los no desenvolvimento de uma página web. Percebemos também quão importante é estudar afundo cada tópico para que possamos estar cada vez mais atualizados no mercado de trabalho.

Conceitos

HTML5 e CSS


Novos conceitos em HTML5 e CSS3 que foram utilizados no desenvolvimento deste projeto:

  • Tag <audio>: Nela podemos setar como parâmetros autoplay e controls, que servem para tocar automaticamente e habilitar os controles do player, respectivamente. O arquivo .mp3 ou mp4, entre outros formatos, é definido na tag pela propriedade “src”;
  • Hover : É usado para selecionar elementos a partir do momento que o cursor passar sobre eles;
  • -webkit-transition : Usado para realçar as imagens da seção 'Galeria';
  • Menu Dropdown usando conceito de sub-listas(lista dentro de lista) e formatação CSS com ajuda do atributo "Visibility".
