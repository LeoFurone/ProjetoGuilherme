Objetivos do grupo na Disciplina

Durante a disciplina, o nosso grupo aprenderá sobre HTML, CSS, JavaScript e PHP. O nosso objetivo é aprender esses tópicos e conseguir aplicá-los no desenvolvimento de uma página web. Percebemos também quão importante é estudar afundo cada tópico para que possamos estar cada vez mais atualizados no mercado de trabalho.

Aqui estão alguns links que geraram interesse do grupo:

  • TUTORIAL HTML - W3Schools
  • TUTORIAL CSS - W3Schools
  • TUTORIAL JAVASCRIPT - W3Schools
  • TUTORIAL PHP - W3Schools

Trabalhos relevantes do Grupo

Os integrantes realizaram trabalho em sua maioria de teor acadêmico, voltado para as áreas de Economia e Teoria Geral dos Sistemas. Na área de Programação, apresentamos um projeto em C, na matéria de Prog.II. Atualmente o grupo está desenvolvendo trabalhos relacionados a Banco de Dados, Análise de Sistemas, e Programação para a Web.
